# 词条自动化获取脚本

# Install
1. python3.9 及以上
2. numpy, pandas, openpyxl, requests  
`pip install --upgrade numpy`  
`pip install --upgrade pandas openpyxl`  
`pip install requests`

# Use
1. 填入变量
* 填入洗孔的 seed 和 index:  
ChangeAbility_seed = ""  
ChangeAbility_index = ""  
* 填入装备的 seed 和 index:  
RandomMainAbility_seed = ""  
RandomMainAbility_index = ""  
2. 运行  
`python get_entries.py`
